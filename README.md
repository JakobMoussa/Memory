🃏 Memory Card Game

A two-player memory card game built with TypeScript and SASS — a personal learning project focused on writing scalable, well-structured frontend code.


🎮 How to Play

Two players take turns flipping cards on the board. On each turn, a player flips two cards:


Match → the player keeps their turn and flips again
No match → the cards are flipped back and it's the other player's turn


The game ends when all pairs have been found. The player with the most matched pairs wins!
